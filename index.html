ACV: Algorithmic Action-Cam Video Generator Using Vehicle Telemetry

Developed the underlying architecture for action camera integration, focusing on concurrent multi-camera video processing, secure data handling, and real-time visual overlays using the Apple SDK and AVFoundation. The algorithm ingests vehicle telemetry data (speed, G-force, drift angle, steering angle, brake torque) and video frames post-recording to automatically generate a shortened, high-engagement Action Cam Video highlight reel.

Designed a framework enabling loosely-coupled multi-product systems through a unidirectional communication architecture. The Parent module orchestrates all child module interactions — child modules remain fully unaware of each other, eliminating direct references and preventing unintended cross-module coupling.
